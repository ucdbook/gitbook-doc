1.1.1. Gitbook 构建

安装gitbook-cli
$ npm install gitbook-cli -g
下载“帮助手册”项目
$ git clone http://gitlab.tf56.lo/yanrongtian/gitbookNote.git
安装gitbook依赖
$ cd gitbookNote
$ gitbook install
启动服务
$ gitbook serve
发布构建
$ gitbook build
